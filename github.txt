Criando chave de segurança, abrir GitBash
> ssh-keygen -t rsa -C "master_bmendes@banparanet.com.br"

Transformar pasta em um repositório Git:
> cd /pasta
> git init

Adicionando uma origin (sincronizando)
> git remote add origin git@github.com:bmendes95/testes.git

a) Clonando projeto do Git
> cd /pasta
> git clone https://github.com/bmendes95/testes.git

b) Controle de Versão
> git tag
> git checkout v0.1

c) O que mudou nos arquivos
> git diff v0.1 v0.2

d) Quais alterações foi feita
> git status

e) Untrack não é monitorado pelo Git
> git add arquivo.txt

f) Commit
> git commit -m "Versão 1 do projeto, melhorias realizadas."  

g) Envia pro repositório remoto
> git push -u origin master  

g) Quem alterou o arquivo
> git blame index.html





Em resumo:
1 - clona o projeto 
> git clone https://github.com/bmendes95/testes.git

2 - cria uma branch
> git branch bug-123

3 - desenvolve

4 - commit 
> git commit -m "Versão 02"

5 - faz merge da "master" pra tua branch atual
> git config --global merge.tool vimdiff
